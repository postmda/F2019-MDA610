HTML Introduction

HTML Documents

All HTML documents start with a document type declaration <!DOCTYPE html>. An HTML document is enclosed by <html> and </html>. The head element, denoted by <head> and </head>, contains the metadata, which may include the title for the document. The visible part of an HTML document is between <body> and </body>.

Basic Elements of an HTML Page

An HTML element is a component of an HTML document. Most HTML elements are written within an opening tag and a closing tag with content in between.

The basic elements include:

  • Text headers, denoted by tags h1, h2, h3, h4, h5, and h6
  • Paragraphs, denoted by the p tag
  • Links, denoted by the a tag
  • Images, denoted by the img tag
  • Dividers, denoted by the div tag
  • Text spans, denoted by the span tag

Elements can also contain attributes that define additional properties. The most common attributes are:

  • id - the unique id of an element in a page
  • class - the CSS class of an element

Internal CSS

Internal CSS styles are defined within the <style> element inside the <head> section. A CSS-rule set consists of a selector and a declaration block.

A selector finds the HTML elements you would like to style. The element selector selects HTML elements by name, such as p, h1, or li. The id selector and the class selector use, respectively, the id attribute and the class attribute to select an element. An id selector precedes an id of the element with a hash (#) character. To select elements of a specific class, write a period (.) character, followed by the class name.
